# my-dotfiles


● tmux 操作一覧

  すべて prefix C-a（Ctrl+a）の後にキーを押す。IMEは自動でoffになる。

  ペイン（画面分割）
  ┌─────────────┬─────────────────────────────────┐
  │    キー     │              動作               │
  ├─────────────┼─────────────────────────────────┤
  │ C-a |       │ 左右に分割                      │
  ├─────────────┼─────────────────────────────────┤
  │ C-a -       │ 上下に分割                      │
  ├─────────────┼─────────────────────────────────┤
  │ C-a h/j/k/l │ ペイン移動（左/下/上/右）       │
  ├─────────────┼─────────────────────────────────┤
  │ C-a H/J/K/L │ ペインリサイズ（Shift+hjkl）    │
  ├─────────────┼─────────────────────────────────┤
  │ C-a z       │ ペインをズーム（全画面化/戻る） │
  ├─────────────┼─────────────────────────────────┤
  │ C-a x       │ ペインを閉じる                  │
  └─────────────┴─────────────────────────────────┘
  ウィンドウ（タブ）
  ┌─────────┬────────────────┐
  │  キー   │      動作      │
  ├─────────┼────────────────┤
  │ C-a c   │ 新規ウィンドウ │
  ├─────────┼────────────────┤
  │ C-a n   │ 次のウィンドウ │
  ├─────────┼────────────────┤
  │ C-a p   │ 前のウィンドウ │
  ├─────────┼────────────────┤
  │ C-a 1-9 │ 番号で切替     │
  └─────────┴────────────────┘
  コピーモード（Vim操作）
  ┌───────────┬────────────────────────────┐
  │   キー    │            動作            │
  ├───────────┼────────────────────────────┤
  │ C-a [     │ コピーモード開始           │
  ├───────────┼────────────────────────────┤
  │ v         │ 選択開始                   │
  ├───────────┼────────────────────────────┤
  │ V         │ 行選択                     │
  ├───────────┼────────────────────────────┤
  │ C-v       │ 矩形選択                   │
  ├───────────┼────────────────────────────┤
  │ y / Enter │ コピー（クリップボードへ） │
  ├───────────┼────────────────────────────┤
  │ Escape    │ キャンセル                 │
  ├───────────┼────────────────────────────┤
  │ h/j/k/l   │ カーソル移動               │
  ├───────────┼────────────────────────────┤
  │ w/b/e     │ 単語移動                   │
  ├───────────┼────────────────────────────┤
  │ 0 / $     │ 行頭/行末                  │
  ├───────────┼────────────────────────────┤
  │ gg / G    │ 先頭/末尾                  │
  ├───────────┼────────────────────────────┤
  │ /         │ 検索                       │
  └───────────┴────────────────────────────┘
  マウスドラッグでもコピー可。

  セッション
  ┌─────────┬────────────────────────────────────────┐
  │  キー   │                  動作                  │
  ├─────────┼────────────────────────────────────────┤
  │ C-a d   │ デタッチ（セッション維持したまま離脱） │
  ├─────────┼────────────────────────────────────────┤
  │ C-a C-a │ シェルにCtrl-aを送信                   │
  └─────────┴────────────────────────────────────────┘
  セッション復帰（ターミナルから）

  tmux attach        # 最後のセッションに復帰
  tmux ls            # セッション一覧
  tmux new -s 名前   # 名前付きセッション作成

  マウス

  - クリック: ペイン切替
  - ドラッグ: ペインリサイズ / テキスト選択→コピー
  - ホイール: スクロール（自動でコピーモード）
