# tmux 操作一覧 (prefix = `C-a`)

`C-a` を押すとIMEが自動でOFFになり、コマンド待ちになる。

## ペイン

| キー | 動作 |
|------|------|
| `C-a \|` | 横に分割 |
| `C-a -` | 縦に分割 |
| `C-a h/j/k/l` | ペイン移動（左/下/上/右・連続可） |
| `Alt+h/j/k/l` | ペイン移動（prefix不要） |
| `C-a H/J/K/L` | ペインリサイズ |
| `C-a z` | ズーム（全画面切替） |
| `C-a x` | ペインを閉じる |

## ウィンドウ

| キー | 動作 |
|------|------|
| `C-a c` | 新規ウィンドウ |
| `C-a n / p` | 次 / 前のウィンドウ |
| `C-a 1-9` | 番号で切替 |
| `C-a a` | 直前のウィンドウに戻る |

## コピーモード（Vim操作）

| キー | 動作 |
|------|------|
| `C-a [` | コピーモード開始 |
| `v` | 選択開始 |
| `V` | 行選択 |
| `C-v` | 矩形選択 |
| `y` / `Enter` | コピー（クリップボードへ） |
| `Escape` | キャンセル |
| `h/j/k/l` | カーソル移動 |
| `w/b/e` | 単語移動 |
| `0` / `$` | 行頭 / 行末 |
| `gg` / `G` | 先頭 / 末尾 |
| `/` | 検索 |

マウスドラッグでもコピー可。

## セッション

| キー | 動作 |
|------|------|
| `C-a d` | デタッチ（セッション維持したまま離脱） |
| `C-a C-a` | シェルにCtrl-aを送信 |
| `C-a C-s` | セッション保存（レイアウト+ペイン+作業ディレクトリ） |
| `C-a C-r` | セッション復元 |

15分ごとに自動保存。tmux起動時に自動復元。PC再起動しても復帰可能。

## レイアウト（プリセット）

| キー | レイアウト |
|------|-----------|
| `C-a M-1` | 均等横並び |
| `C-a M-2` | 均等縦並び |
| `C-a M-3` | メイン左+右に小ペイン |
| `C-a M-4` | メイン上+下に小ペイン |
| `C-a M-5` | タイル状 |

## ヘルプ

| キー | 動作 |
|------|------|
| `C-a ?` | チートシートをポップアップ表示 |
| `C-a Space` | which-key メニュー |
| `C-a R` | 設定リロード |

## ターミナルから

```bash
tmux attach        # 最後のセッションに復帰
tmux ls            # セッション一覧
tmux new -s 名前   # 名前付きセッション作成
```

## マウス

- **クリック**: ペイン切替
- **ドラッグ**: ペインリサイズ / テキスト選択→コピー
- **ホイール**: スクロール（自動でコピーモード）
